Comparison Operators

Comparison operators are used to compare two values and return a boolean result (true or false). They are commonly used in conditional statements and loops to control the flow of a program. Here are some of the most common comparison operators:

  • Equal to (==): Checks if two values are equal. Example: 5 == 5 returns true.
  • Not equal to (!=): Checks if two values are not equal. Example: 5 != 3 returns true.
  • Greater than (>): Checks if the left value is greater than the right value. Example: 5 > 3 returns true.
  • Less than (<): Checks if the left value is less than the right value. Example: 3 < 5 returns true.
  • Greater than or equal to (>=): Checks if the left value is greater than or equal to the right value. Example: 5 >= 5 returns true.
  • Less than or equal to (<=): Checks if the left value is less than or equal to the right value. Example: 3 <= 5 returns true.
